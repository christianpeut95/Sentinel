# Apple-Inspired UI Mockups for Surveillance System

> Clean, minimal aesthetic with maximum information density

## Design Principles

### Apple Aesthetic Elements
- **Typography**: SF Pro Display/Text, -apple-system font stack
- **Colors**: System grays, SF Symbols, subtle gradients
- **Spacing**: 8px grid system (4, 8, 12, 16, 24, 32, 48)
- **Corners**: 8-12px border radius
- **Shadows**: Subtle, layered (0 2px 8px rgba(0,0,0,0.04))
- **Whites pace**: Generous but purposeful
- **Icons**: SF Symbols style - simple, recognizable

### Information Density Strategy
- **Cards over tables** where possible, but tables for lists
- **Inline editing** for quick updates
- **Collapsible sections** with indicators
- **Smart grouping** of related information
- **Visual hierarchy** through size, weight, color
- **Status indicators** as colored dots (8px)
- **Sticky headers** for context while scrolling

## Interaction Patterns

### Hover States
- **Cards**: Slight lift (0 4px 12px rgba shadow)
- **Buttons**: Darken by 10%
- **Table Rows**: Light gray background (#F9F9F9)

### Active/Focus States
- **Primary elements**: Blue ring (2px, #007AFF)
- **Danger elements**: Red ring (2px, #FF3B30)

### Loading States
- **Skeleton screens**: Gray placeholders that pulse
- **Spinners**: Small (16px) for inline, large (32px) for full page

### Transitions
- **Standard**: 200ms ease-in-out
- **Quick**: 100ms ease-in-out (hovers)
- **Slow**: 300ms ease-in-out (page transitions)

## Accessibility Considerations

4. **ARIA Labels**: Proper roles and labels for screen readers
5. **Contrast**: Maintain WCAG AA standards (4.5:1 minimum)
