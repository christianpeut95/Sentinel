- **Survey Section**: Light gray background (`bg-light`)
- **Checkbox**: Bootstrap switch style
- **Radio Buttons**: Clear options with descriptions
- **Dropdown**: Full-width select with categories
- **Textarea**: Monospace font for JSON

## ?? State Transitions

### **Initial State**
- Survey section collapsed
- Checkbox unchecked
- Library option selected (default)

### **Survey Enabled**
- Survey section visible
- Options expanded
- Library dropdown shown
- Custom textarea hidden

### **Custom Selected**
- Library dropdown hidden
- Custom textarea shown
- JSON ready for input

### **Form Reset**
- All fields cleared
- Survey section collapsed
- Defaults restored

## ?? Accessibility

### **Keyboard Navigation**
- Tab through all fields
- Space to toggle checkbox
- Arrow keys in dropdowns
- Enter to submit

### **Screen Readers**
- Labels for all inputs
- ARIA labels on icons
- Descriptive link text
- Status announcements

### **Visual Indicators**
- Required fields marked with `*`
- Focus states on inputs
- Disabled state styling
- Color-blind friendly
