## ?? Auto-Assignment Algorithm Flow

```
???????????????????????????????????????????????????????????
?              AUTO-ASSIGNMENT PROCESS                     ?
???????????????????????????????????????????????????????????

Worker clicks "Get Next Task"
         ?
         ??? Check Worker Status
         ?     ?
         ?     ??? Is worker available? ?????? NO ?? Return "Unavailable"
         ?     ?                          ?
         ?     ?                          YES
         ?     ?                          ?
         ?     ??? Check current task count
         ?     ?     ?
         ?     ?     ??? At capacity? ??????? YES ?? Return "At capacity"
         ?     ?     ?                    ?
         ?     ?     ?                    NO
         ?     ?     ?                    ?
         ?     ?     ??? Get unassigned tasks
         ?     ?
         ?     ??? Filter tasks
         ?           ?
         ?           ??? Where IsInterviewTask = true
         ?           ??? Where AssignedToUserId = null
         ?           ??? Where Status = Pending
         ?           ??? Optional: Match language
         ?
         ??? Sort tasks
         ?     ?
         ?     ??? By EscalationLevel (highest first)
         ?     ??? By Priority (Urgent > High > Med > Low)
         ?     ??? By CreatedAt (oldest first)
         ?
         ??? Select first task
         ?     ?
         ?     ??? If language matching enabled:
         ?     ?     ?
         ?     ?     ??? Check worker's languages
         ?     ?     ??? Match task's LanguageRequired
         ?     ?     ??? If no match, try next task
         ?     ?
         ?     ??? If round-robin only:
         ?           ?
         ?           ??? Take first task from sorted list
         ?
         ??? Assign task
         ?     ?
         ?     ??? Set AssignedToUserId = worker
         ?     ??? Set AssignmentMethod = AutoRoundRobin or AutoLanguageMatch
         ?     ??? Set AutoAssignedAt = now
         ?     ??? Save to database
         ?
         ??? Return task to worker
               ?
               ??? Worker dashboard displays task
```

```

         ?

         ?     • TaskId
         ?     • Outcome
         ?     • Notes
         ?     • Timestamp
         ?     • WorkerId
         ?
         ??? Increment CurrentAttemptCount
         ?     • Was: 2/3
         ?     • Now: 3/3
         ?
         ??? Check if CurrentAttemptCount >= MaxCallAttempts
         ?     ?
         ?     NO ?? Keep task assigned
         ?     ?       ??? Status: In Progress
         ?     ?
         ?     YES
         ?     ?
         ?     ??? ESCALATE
         ?     ?     ?
         ?     ?     ??? Increment EscalationLevel (0 ? 1)
         ?     ?     ??? Set Priority = High

         ?     ?     ??? Set Status = Pending

         ?     ?

         ?           ?
```

## ?? Responsive Design Notes

### Worker Dashboard:
- **Desktop:** Full layout with side-by-side panels
- **Tablet:** Stacked panels, full functionality
- **Mobile:** Simplified view, essential buttons prominent

### Supervisor Dashboard:
- **Desktop:** Multi-column tables
- **Tablet:** Scrollable tables
- **Mobile:** Cards instead of tables, tap to expand
